# ModbusTCPDemo

### 项目说明
本项目是使用ModbusTCP协议进行通信的一个Demo，使用C#开发，开发工具VS2022，开发环境 .NET6。

完成了协议各个功能码报文的拼接、发送、接收、校验和解析. 

也可以使用通信库[NModbus4.NetCore](https://github.com/HakamFostok/NModbus4.NetCore)

## 功能

- [x] 读取输出线圈
- [x] 读取输入线圈
- [x] 读取保持寄存器
- [ ] 读取输入寄存器
- [ ] 写入单个输出线圈
- [ ] 写入多线圈
- [ ] 写入单寄存器
- [x] 写入多寄存器
